# Agent Scripts

This folder contains scripts created and used by the agent.

## Structure

- Scripts are created by the agent during task execution
- Each script has a unique name describing its purpose
- Scripts can be `.bat`, `.ps1`, `.py`, etc.

## Security

- Scripts execute in a sandboxed environment
- Terminal capability must be enabled for script execution
- All script executions are logged

## Example Scripts

Scripts will be created here as the agent works on tasks.
